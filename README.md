# Cryptocurrency Design

A responsive cryptocurrency website that displays real-time prices for Bitcoin, Ethereum, and Dogecoin. The site features a clean design with navigation, market information, and an interactive coin list.

## Features

- Real-time cryptocurrency price display
- Responsive design for web and mobile
- Navigation menu with market, features, white papers, and about sections
- Interactive coin list showing Bitcoin, Ethereum, and Dogecoin

## Live Demo

View the live website at: https://iam269.github.io/Cryptocurrency-Design/

## Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
